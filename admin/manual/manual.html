Introduction

If you want to start your own online auction site, then this software is for you. This manual deals with all administration functions. With well over 200 user-friendly features and administrative features, AJAuction is a powerful PHP/MySQL powered domain name auction script. Built for speed, features include the ability to bulk load domain. In addition we have taken a fresh approach to creating a simple, easy to learn administration panel that will have you operating like a pro in less than 30 minutes. This section discusses the organization of the manual

Installation Introduction

Installation guide are available in the doc directory of the product’s compressed file. The following instruction manual can be found in the “install.txt” which can be found in the installation package.

Step 1: Make sure AJAuction files are uploaded in the correct path in the server.
Step 2: Set the permission to the include folder, images folder to 777 (read, write, execute) and storefronts
folder permission for the scoping directory and include directory.
Step 3: Create a new database for the script.
Step 4: Create a new username and password for the script.
Step 5: Make sure you grant all permissions in the database for the created user.
Step 6: Please provide the following information correctly to install the script.
       i. Database Host name – usually local host for most of the system, else give the host name correctly.
       ii.Database Name – Please give the name of the database you have created for our script.
       iii. Database Username – Please give the username you have created for our script.
       iv. Database Password – Please give the password you have created for our script.

Creating Cpanel

cpanel address will be like this, http://www.yoursitename.com/cpanel .Then you will be prompted for username and password, if the information is correct you will be taken to home page of the c panel. Creating Database, Database username and Password Click MySQL Databases link from the home of cpanel. You will be taken to a page where you could find the list of already existing database. Enter the Database Name there and press. Create MySQL database Button, a new database will be created in the database name you gave, prefixed by the cpanel username.

For example: If the cpanel is your username, then the database name will look like yourusername_auction.

Setting permission for Database

For granting permissions for the database, select the user you have created, select MySQL Database and select ALL in privileges then click Grant permissions button.

Note: If you use any other server control panel other than cpanel, please follow instruction given by your service provider for creating database.

To access Administration Panel

To access the administration control panel after the software is installed, browse the www.yoursite.com/admin page. The default admin login is admin/admin….. is recommended to change the admin username and password under “change password”.
